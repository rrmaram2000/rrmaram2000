## About

I build adaptive signal processing methods that learn directly from data. My research bridges **classical wavelet theory** with **modern deep learning**, creating tools that preserve the mathematical elegance of wavelets while gaining the flexibility of neural networks.

Currently developing the first open-source MATLAB implementation of learnable wavelet layers â€” enabling researchers to integrate adaptive multiresolution analysis into their medical imaging pipelines.

## Research Focus

> **Learnable Wavelets for Medical Imaging**
>
> Traditional wavelets use fixed, analytically-defined filters. My approach optimizes wavelet coefficients during training, allowing the decomposition to adapt to specific imaging tasks â€” whether that's tumor detection in CT, artifact reduction, or feature extraction for radiomics.

**Key Contribution:** First open-source MATLAB implementation bridging wavelet signal processing with deep learning frameworks.
